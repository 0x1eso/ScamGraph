## 1. 판정 정책 요약

- **판정 원칙**: 순수 규칙 기반, AI 없음. 모든 판정에 근거(`reasons`) 첨부(설명 가능성).
- **등급 체계**: `safe`(안전, 근거충분) · `caution`(주의) · `warning`(경고) · `danger`(확인된 위협). `safe` 는 "안전 보장"이 아니라 "아직 악성 미확인".
- **신고 승격 정책**(요지): 신고 1건=저장만 → 다수=표시 → 신고+강신호=위험후보 → 신뢰 feed=확인 → 운영자+복수 강증거=blocklist. 상세: [`abuse-defense.md`](abuse-defense.md) §3.
